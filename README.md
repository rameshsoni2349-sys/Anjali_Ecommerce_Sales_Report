# Anjali_Ecommerce_Sales_Dashboard 
Project : Ajali Ecommerce Dashboard
Sales Dashboard Project – Power BI

• Created an interactive dashboard using multiple visualizations such as Cards, Donut Charts, Clustered Column Charts, Clustered Bar Charts, and Slicers.

• The dashboard provides monthly and quarterly sales reports for better business analysis.

• Analyzed key metrics including:

* Profit by Month
* Profit by Category
* Quantity by Category
* Payment Mode by Category
* Amount by Customer Name
* Amount by State

• Added KPI indicators to track overall business performance and insights.
